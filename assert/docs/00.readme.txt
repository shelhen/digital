二、数据清洗原则
      1.直接从数据库整理得到的数据可能存在一些问题，这里对数据进行一下清洗
        - 大量异常值，远离于正常值的正态分布以内
        - 数据不够全面，由于删除了一些空值，因此可能造成部分企业部分时间内出现缺失值，这不利于数据挖掘
        - 有些空值完全经过填充而得，不知道是否会对结果有影响
      2.空值替换
       - 替换为同股票代码下的平均值
       - 填充为0
       - 填充为前后均值
       - 预测填充
      3.异常值
      检测出异常值，填充(同上)异常值

三、企业绩效数据的清洗与处理
    1.删除期间存在过st的所有企业
    2.使用线性插值法填充数据
    3.对于部分数据，有些空值代表为0，可以直接填充
    4.删除缺失数值超过80%的指标——找不到数据或找到的数据太少
    5.删除空值，对异常数据进行缩尾处理
    6.构建新的指标，如企业年限，行业类别等等
    7.最终获得有效指标56列
